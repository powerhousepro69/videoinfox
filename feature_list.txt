
Videoinfox v5.9.44 Features: What each feature does and its location.

* = Home Screen

* Default dirs 1-5
* Find Video - Recuresively search from the current directory
* Show All - List all recursive from the current directory.
* Clear Results - Clear the find or show all results from the home screen.
* Find Dir -  Enter number to change directory.
* Last Download - Play last download. Applies to play clipboard, download only and download choose.
* Playlist Find - Locate videos by keyword. Search all playlists and the default directories index.
* Playlist Add - Add videos to loaded playlist from recursive search results or show all.
* Make Playlist - Make a new playlist from all results on the home screen.
* Play Clipboard - Play a local file or a download URL from clipboard and play. Writes to the played list.
* Download Only - Download URL in clipboard. Don't play.
* Play Directory - Play the current directory. One level deep.
* Play - Play only the line number entered. Writes to played list.
* Autoplay - Enter line number to start autoply from anywhere in the list.                         
* Info - Get codec, resolution and duration on line number entered.
* Clrclp - Clear out Clipboard
* Help - Help on most features.

Command line options
 
Note: Videoinfox won't accept multiple options. Only one option per instance.


   -h  This help screen.

   -v  Display the version number.

   -a  Write video URL in the clipboard to the Default List. Instructions: Yt-dlp >>> Download List >>> Help

   -l  Write video URL in the clipboard to the User Set List. Instructions: Yt-dlp >>> Download List >>> Help

   -d  Play Directory. Autoplay video files in the current directory. Change intermission time in Settings

   -z  Make a playlist from a recursive video listing in the current directory. The Playlist will be saved in
       ~/.config/videoinfox/playlists which makes the playlist/index available to videoinfox's feature set.

   -u  This option can be used with your own ready to download url list that can be any name. Videos will be
       downloaded to the directory of the list's location. Example: videoinfox -u mylist  Only enter the list
       filename. No Paths. The list must be recursively reachable from the current directory. THE FIRST LINE
       IN THE LIST MUST BEGIN WITH  http  TO BE PROCESSED. Any files processed with this option will delete all
       lines that don't start with http, all blank lines and all duplicate lines.
